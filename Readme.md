\# Hydraulic Condition Monitoring



Aplicación desarrollada en Streamlit para predecir la estabilidad de un sistema hidráulico utilizando un modelo Random Forest.



\## Variables utilizadas



\- Sensores de presión

\- Sensores de temperatura

\- Sensores de flujo

\- Potencia

\- Vibración

\- Eficiencia



\## Ejecución local



streamlit run app.py
